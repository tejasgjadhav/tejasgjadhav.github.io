Live dashboard tracking Amazon Best Seller Ranks for [[tejas-jadhav]]'s **5 KDP books**
(full catalog: [[kdp-books]]) across **3 formats and 15 countries**, updated daily via
GitHub Actions and served with GitHub Pages. Structure: `index.html` dashboard, `scripts/`
for the daily rank scrape, `data/books.json` + `data/rankings.json` for catalog and
accumulated rank history, `images/` for covers.

A sixth title, [[dotnet-architect-book]], is in build and not yet in the catalog.

Part of [[files-repo]].
